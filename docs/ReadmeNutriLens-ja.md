
## 📣 NutriLens AI（AI 食事・栄養分析）

食事の写真を撮るだけで、NutriLens AI が料理を識別し、カロリー、たんぱく質、脂質、炭水化物、食物繊維、ビタミン、ミネラルなどを含む栄養バランスを分析します。一般モードでは、料理の概要、主な食材、実用的な栄養アドバイスをわかりやすく提供し、プロフェッショナルモードでは、詳細な一食分の重量推定と包括的な栄養レポートを生成します。日々の食事を、ひと目でより簡単に、より楽しく理解できるようにします。

---

## 📖 ユーザーガイド：NutriLens AI

NutriLens AI は、食品、食事、飲み物の写真を AI で分析し、栄養成分を推定する Web ベースのサービスです。定食、複数の料理、トレーに並べられた食事、弁当、スープ、ご飯、副菜、デザート、飲み物など、幅広い食事画像を分析できます。

1. **言語を選択する**  
   「日本語」または「English」のいずれかを選択します。ブラウザ設定に基づいて、既定の言語が自動的に選択される場合があります。

2. **分析モードを選択する**  
   必要に応じて、「一般モード（一食分推定と概要）」または「プロフェッショナルモード（詳細分析と検証記録）」のいずれかを選択します。

3. **画像をアップロードする**  
   「食事写真を選択」ボタンを使用して、食品、食事、または飲み物が写った写真を選択します。PNG、JPEG、WebP、HEIC ファイルに対応しています。推奨される最大ファイルサイズは 5 MB です。

4. **分析を開始する**  
   「食事を分析」を選択し、結果が表示されるまで待ちます。一般モードでは簡潔な概要を提供し、プロフェッショナルモードではより詳細な栄養分析を行います。

### 🔍 分析モード

* **一般モード（一食分推定と概要）**  
  このモードでは、料理名を推定し、目に見える食材をおよそ 3〜5 種類識別します。また、実用的な栄養アドバイスと食物繊維量のおおまかな推定も提供します。食事の主な栄養的特徴をすばやく理解するのに適しています。

* **プロフェッショナルモード（詳細分析と検証記録）**  
  このモードでは、画像内の可能性が高い食材を識別し、1 食あたりの重量、想定される誤差範囲、信頼度を推定します。そのうえで、エネルギー、たんぱく質、脂質、炭水化物、食物繊維、脂肪酸、ビタミン、ミネラル、その他の栄養素をより詳細に計算します。スプーン、箸、茶碗、皿の縁などの物体が見える場合、アプリはそれらをサイズの参照として使用し、より現実的な重量推定を行うことがあります。

### 📊 プロフェッショナルモードで利用できる主な情報

* 識別された食材と推定重量を含む推定根拠
* 識別された全食材における平均信頼度
* エネルギー、たんぱく質、脂質、炭水化物
* 食物繊維：総量、水溶性、不溶性
* 脂質およびたんぱく質のエネルギー比率
* 飽和脂肪酸、n-3 系脂肪酸、n-6 系脂肪酸、および n-3/n-6 比
* ビタミン A、B₁、B₂、C、D、および葉酸
* カルシウム、リン、カルシウム／リン比、鉄、マグネシウム、亜鉛、カリウム
* 調理に伴う補正で使用した前提条件

### 💾 結果の保存

プロフェッショナルモードでは、必要に応じて分析結果を Markdown 形式で保存できます。結果画面で Markdown エクスポート用パスワードを入力し、「MD として保存」を選択します。ファイル名は自動的に生成され、日付と連番が含まれます。

### ⚠️ 重要な情報

* 本サービスは、食事と栄養バランスに関する一般的な情報を提供することを目的としています。医学的診断、治療、または個別の食事指導を提供するものではありません。
* 糖尿病、腎疾患、食物アレルギー、食事制限がある場合、妊娠中の場合、またはその他慎重な食事管理が必要な場合は、医師または管理栄養士に相談してください。
* 画像に基づく重量および栄養素の推定には誤差が伴います。奥行き、隠れた食材、調味料、油、ソース、調理方法は、写真だけでは正確に判断できない場合があります。
* アプリは、食品、食事、飲み物が含まれていない画像を分析できません。食べ物が見やすい鮮明な写真を撮影してください。
* アップロードされた画像は分析のみに使用されます。個人情報や不要な背景情報が画像に含まれないようにしてください。
* プロフェッショナルモードでは、コンテンツ保護のため、結果画面からの右クリック、コピー、印刷が制限される場合があります。記録を残す必要がある場合は、Markdown エクスポート機能を使用してください。
* 過度なトラフィックを防ぐため、短時間での繰り返し利用が制限される場合があります。利用制限のメッセージが表示された場合は、時間をおいてから再試行してください。
